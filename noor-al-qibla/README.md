# Noor Al Qibla Website

Official website for Noor Al Qibla - Islamic Prayer Times & Qibla Direction app.

## About

Noor Al Qibla is a comprehensive Islamic application that provides:
- Accurate prayer times
- Qibla direction
- Hijri calendar
- Digital Tasbih
- Prayer notifications
- Home screen widgets

Available in 9 languages: English, Arabic, Persian, Urdu, Turkish, Indonesian, Hindi, French, and Spanish.
